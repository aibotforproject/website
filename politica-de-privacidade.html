Política de Privacidade

Data da disponibilidade: 01/05/2025

1. Introdução

Esta Política de Privacidade descreve como a ia24hs coleta, usa, armazena e protege as informações pessoais de nossos clientes empresariais e seus usuários finais. Esta política se aplica exclusivamente a empresas que contratam nossos serviços de inteligência artificial e automação.

2. Coleta de Dados

2.1 Dados dos Clientes

A ia24hs coleta somente os dados necessários para identificação do cliente para elaboração do contrato, incluindo informações básicas da empresa contratante.

2.2 Dados dos Usuários Finais

Os dados dos clientes dos clientes são armazenados em nossos servidores conforme a necessidade dos clientes que contratam nossas aplicações, assim como de terceiros que por necessidade do negócio possuem contato quando necessário, como exemplo serviços de inteligência artificial.

2.3 Formulários e Eventos

Não coletamos dados através de formulários específicos no site ou eventos especiais.

3. Finalidades do Processamento

Os dados são utilizados exclusivamente para a finalidade do negócio do cliente ou para criação de contexto para utilização dos serviços de inteligência artificial. Não realizamos análises de desempenho, marketing direcionado ou melhorias no serviço com base nos dados coletados.

3.1 Comunicações de Marketing

Não enviamos comunicações de marketing para nossos clientes.

4. Compartilhamento de Dados

4.1 Serviços de Terceiros

Compartilhamos dados com os seguintes serviços de inteligência artificial: OpenAI, Deepgram e Gemini (Google).

4.2 Armazenamento Internacional

Utilizamos serviços de terceiros para armazenamento ou processamento de dados fora da Europa, especificamente os serviços de inteligência artificial mencionados acima.

5. Retenção e Exclusão de Dados

5.1 Período de Retenção

Os dados pessoais dos clientes são retidos conforme contrato com o cliente. Os dados são dos clientes e a ia24hs não os utiliza para nenhuma finalidade além da prestação dos serviços contratados.

5.2 Solicitações de Exclusão

As solicitações de exclusão de dados são processadas conforme necessidade do cliente ou definição de legislação aplicável.

5.3 Arquivamento e Anonimização

O arquivamento ou anonimização de dados é realizado conforme necessidade do cliente ou definição de legislação.

6. Segurança e Local de Armazenamento

6.1 Medidas de Segurança

O acesso aos dados por entidades externas são protegidas e somente credenciados possuem acesso quando definido pelo cliente.

6.2 Localização dos Dados

Não há outros locais de armazenamento além da Europa para os dados principais dos clientes.

7. Transferências Internacionais

Como os dados são armazenados na Europa, não possuímos cláusulas contratuais ou medidas específicas para cumprir a LGPD em relação a transferências internacionais, uma vez que nossas operações principais estão localizadas na Europa.

8. Público-Alvo

Os serviços da ia24hs são direcionados exclusivamente a empresas. Não atendemos indivíduos ou menores de idade, o que impacta diretamente nossa política de privacidade e coleta de dados.

9. Direitos dos Titulares dos Dados

Conforme a LGPD, os titulares dos dados têm direito de acesso, correção, exclusão, portabilidade e oposição ao tratamento de seus dados pessoais. Para exercer esses direitos, entre em contato conosco através dos canais disponibilizados.

10. Alterações na Política

Podemos atualizar esta Política de Privacidade a qualquer momento. Notificaremos sobre quaisquer mudanças através dos meios de comunicação fornecidos ou através de nosso site.

11. Contato

Para dúvidas, esclarecimentos ou exercício de direitos relacionados aos seus dados pessoais, entre em contato conosco através do e-mail aibotforproject@gmail.com.
